```
                        в”Җ в”Җ в”Җ в”Җ в”Җ в”Җ в”Җ в”Җ в”Җ в”Җ в”Җ в”Җ в”Җ в”Җ в”Җ в”Җ в”Җ в”Җ
 comprehension-led   вҸӯ skipped     COMP_* scores      вң… MCQs (optional)
 discussion-led      вқҢ none       DISC_* scores       вқҢ none
 coaching-led        вқҢ none       COACH_* scores      вқҢ none
```

**Key insight:** The two paths are additive. Knowledge courses already get session scoring from the pipeline вҖ” session-embedded measurement just adds *learning outcome* scoring for courses that can't use pre/post tests.

## Session-Embedded Score Types

| Profile | Scores extracted per session |
|---------|----------------------------|
| **comprehension-led** | COMP_THEME В· COMP_INFERENCE В· COMP_EVIDENCE В· COMP_RECALL |
| **discussion-led** | DISC_PERSPECTIVE В· DISC_ARGUMENT В· DISC_SHIFT В· DISC_REFLECTION |
| **coaching-led** | COACH_CLARITY В· COACH_ACTION В· COACH_AWARENESS В· COACH_FOLLOWUP |

## Decision Record

- **2026-04-05 (Boaz):** Comprehension courses skip pre-tests. Measurement embedded in sessions.
- **2026-04-05 (Implementation):** Comprehension MCQs tagged `assessmentUse: POST_TEST`. Pre-test builder excludes them.
- **2026-04-05 (Epic):** Session-Embedded Learning Measurement вҖ” 3 non-knowledge profiles, same pipeline infra, different specs. ~19h, 5 stories.
